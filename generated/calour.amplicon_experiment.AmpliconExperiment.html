calour.amplicon_experiment.AmpliconExperiment¶

class calour.amplicon_experiment.AmpliconExperiment(data, sample_metadata, feature_metadata=None, exp_metadata={}, description='', sparse=True)[source]¶

This class contains the data for a experiment or a meta experiment.

Parameters:

data : numpy.ndarray or scipy.sparse.csr_matrix

The abundance table for OTUs, metabolites, genes, etc. Samples are in row and features in column

sample_metadata : pandas.DataFrame

The metadata on the samples

feature_metadata : pandas.DataFrame

The metadata on the features

description : str

name of experiment

sparse : bool

store the data array in scipy.sparse.csr_matrix or numpy.ndarray

See also

Experiment
