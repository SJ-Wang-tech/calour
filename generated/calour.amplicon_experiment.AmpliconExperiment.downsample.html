calour.amplicon_experiment.AmpliconExperiment.downsample¶

AmpliconExperiment.downsample(exp, field, axis=0, num_keep=None, inplace=False)[source]¶

Downsample the data set.

This down samples all the samples/features to have the same number of samples/features for each categorical value of the field in sample_metadata or feature_metadata.

Parameters:

field : str

The name of the column in samples metadata table. This column should has categorical values

axis : 0, 1, ‘s’, or ‘f’

0 or ‘s’ (default) to filter samples; 1 or ‘f’ to filter features

num_keep : int or None (optional)

None (default) to downsample to minimal group size. int : downsample to num_keep samples/features per group, drop values with < num_keep

inplace : bool (optional)

False (default) to do the filtering on a copy. True to do the filtering on the original Experiment
Returns:

Experiment
