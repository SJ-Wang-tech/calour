calour.io.save¶

calour.io.save(exp, prefix, fmt='hdf5')[source]¶

Save the experiment data to disk.

Parameters:
  • prefix (str) – file path to save to.
  • fmt (str) – format for the data table. could be ‘hdf5’, ‘txt’, or ‘json’.
