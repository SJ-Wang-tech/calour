calour.sorting.sort_by_metadata¶

calour.sorting.sort_by_metadata(exp, field, axis=0, inplace=False)[source]¶

Sort samples or features based on metadata values in the field.

Parameters:
  • field (str) – Name of the field to sort by
  • axis (0 or 1) – sort by samples (0) or by features (1), i.e. the field is a column in sample_metadata (0) or feature_metadata (1)
  • inplace (bool (optional)) – False (default) to create a copy True to Replace data in exp
Returns:
Return type:

Experiment
