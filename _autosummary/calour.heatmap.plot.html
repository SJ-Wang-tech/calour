calour.heatmap.plot¶

calour.heatmap.plot(exp, sample_color_bars=None, feature_color_bars=None, gui='cli', databases=('dbbact', ), color_bar_label=True, **kwargs)[source]¶

Plot the main heatmap and its associated axis.

Parameters:
  • exp (Experiment) – the object to plot
  • sample_color_bars (list, optional) – list of column names in the sample metadata. It plots a color bar for each column. It doesn’t plot color bars by default (None)
  • feature_color_bars (list, optional) – list of column names in the feature metadata. It plots a color bar for each column. It doesn’t plot color bars by default (None)
  • color_bar_label (bool, optional) – whether to show the label for the color bars
  • gui (str, optional) – GUI to use
  • databases (Iterable of str) – a list of databases to access or add annotation
  • kwargs (dict, optional) – keyword arguments passing to heatmap function.
Returns:
Return type:

PlottingGUI
