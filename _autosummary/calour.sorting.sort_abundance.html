calour.sorting.sort_abundance¶

calour.sorting.sort_abundance(exp, field=None, value=None, inplace=False, **kwargs)[source]¶

Sort features based on their abundance in a subset of the samples. This is a convenience wrapper for sort_by_data()

Parameters:
  • field (str or None (default)) – None (default) to sort on all samples, str to sort only on samples matching the field/value combination
  • value (str or list of str or None (default)) – if field is not None, value is the value/list of values so sorting is only on samples matching this list
  • inplace (bool (optional)) – False (default) to create a copy of the experiment, True to filter inplace
Returns:

with features sorted by abundance
Return type:

Experiment
