calour.filtering.filter_ids¶

calour.filtering.filter_ids(exp, ids, axis=1, negate=False, inplace=False)[source]¶

Filter samples or features based on a list index values

Parameters:
  • ids (iterable of str) – the feature/sample ids to filter (index values)
  • axis (int (optional)) – 1 (default) to filter features, 0 to filter samples
  • negate (bool (optional)) – False (default) to keep only sequences matching the fasta file, True to remove sequences in the fasta file.
  • inplace (bool (optional)) – False (default) to create a copy of the experiment, True to filter inplace
Returns:

filtered so contains only features/samples present in exp and in ids
Return type:

Experiment
